ConsoleOutput¶

Default class for all CLI output. Uses STDOUT.

This class is a convenient wrapper around `StreamOutput`. $output = new ConsoleOutput; This is equivalent to: $output = new StreamOutput(fopen('php://stdout', 'w'));

StreamOutput¶

Write output to given stream.

Usage: $output = new StreamOutput(fopen('php://stdout', 'w')); As `StreamOutput` can use any stream, you can also use a file: $output = new StreamOutput(fopen('/path/to/output.log', 'a', false));
