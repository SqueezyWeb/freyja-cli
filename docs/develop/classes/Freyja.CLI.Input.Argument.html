Arguments

$name

string

Argument name.

$mode

integer

Optional. Argument mode: self::REQUIRED, self::OPTIONAL, or self::IS_ARRAY. If null, it is initialized at self::OPTIONAL. Default null.

$description

string

Optional. Argument description text. Default empty.

$default

mixed

Optional. Default value (for self::OPTIONAL or self::IS_ARRAY modes only). Default null.
